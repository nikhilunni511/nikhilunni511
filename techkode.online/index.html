Courses

JAVASCRIPT

You will learn programming fundamentals and the object-oriented concepts using the latest JavaScript syntax. The concepts covered in these lessons lay the foundation for using JavaScript in any environment.

Prerequisites: None

Duration: 18hrs

ES6

Learn about the changes made to the JavaScript programming language in ECMAScript 6 and how to use these new improvements to write better cleaner code. Concepts like spread, arrow, async/await and many more.

Prerequisites: JavaScript

Duration: 12hrs

GIT

This course is designed to be a comprehensive approach to Git, which means no prior knowledge or experience is required but students will emerge at the end with a very solid understanding and hands-on experience.

Prerequisites: None

Duration: 18hrs

DESIGN PATTERN

JavaScript developers frequently interact with design patterns, even unknowingly, when creating applications. Learn the most important patterns like module, prototype, singleton and many more using JavaScript.

Prerequisites: JavaScript, ES6

Duration: 12hrs

ANGULAR 6

In this course, you're going to learn about the core fundamentals of Angular 6 while building an actual app. Concepts like components, routing, services, pipes, RxJS Obervables and many more.

Prerequisites: JavaScript, ES6

Duration: 36hrs

IONIC

Learn Ionic framework for developing apps that are hybrid (that is, built to work across the iOS and Android platforms) using HTML, CSS, and JavaScript. It's popular among Angular developers.

Prerequisites: Angular 6

Duration: 24hrs

VUE 2.0

In this course, you're going to learn about the progressive framework while building an actual app. This course covers concepts like components, routers, props, lifecycles, slots and many more advanced concepts.

Prerequisites: JavaScript, ES6

Duration: 36hrs

React

In this course, you're going to learn about the core fundamentals of React and redux while building an actual app. Concepts like components, react routers, props, lifecycles and many more advanced topics.

Prerequisites: JavaScript, ES6

Duration: 36hrs

Spring Boot

In this course, you're going to create a stand-alone and production ready spring applications using Spring Boot. This course covers concepts like Spring AOP, Microservices, JUnit, Mockito many more advanced topics.

Prerequisites: Basics Java Concepts

Duration: 30hrs
